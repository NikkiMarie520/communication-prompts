# 💬 Communication Prompt Generator

A simple Python tool that helps users structure their thoughts and communicate more clearly.

---

## ✨ Purpose

This project was created to combine my background in psychology with my growing interest in software engineering.

Clear communication can be difficult, especially during emotional situations. This tool provides a simple structure to help users express themselves more effectively.

---

## ⚙️ How It Works

The program asks the user what they are struggling with and provides a guided communication framework.

Example output:
"I feel ___ when ___ because ___. What I need is ___."
